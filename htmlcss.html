CSS: Cascade Style Sheet

  • style < style="color:blue;text-align:center;">: change style
  • better not mix with content of html
  • in header: <style> h1{color:blue;text-align:center;} <\style>: apply to all h1 tags.
  • but can also put in a different file: .css
  • on header:
  • in file.css: h1{color:blue;text-align:center;}
  • Atributes:
    • color, background-color: name, hexadecimal color
    • size, margin, padding: pixels
    • font-family, font-size, font-weight (bold)
    • border: make boxes around elements, bordercollapse=collapse: remove double borders
  • <div id="label">: name it
  • #label {}: apllies to al labels with id "label"
  • .clasname{}:apply to all classes (elements that have class="classname"
  • Selectors:
    • h1,h2{}: apply to both
    • ol li{}: apply to li that are children of ol (and to ol).
    • ol>li{}: apply only to li that are first gen child
    • input[type=text]{}: only when type is meant to be text (in a form for instance)
    • input[type=number]{}: only when type is meant to be number
    • pseudo class:
    • button:hover{}: apply when hovering the button (is a pseudo class)
    • a::before{content:"something"}: substitutes #.
    • p::selection{}: on selecting something
  • Responsive design:
    • media queries: style depend on media.
    • @media print{.screen-only{display:none}}: not shown in printing
    • @media (min-width:500px){}: if viewed in 500 or more px
    • @media (max-width:499px){}: if viewed in 499 or less px
    • <meta name="viewport content="width=device-width">: adapt the viewport to device
    • flexbox:display:flex; : flexibly change the arrange of things.
    • grid: display:grid; control grid of tables
  • bootstrap:
    • divide the page in 12 columns
    • get style from a link in internet
    • class has names: col-3, col-lg-6, alert alert-primary, etc.
    • see: getbootstrap.com
