git

My notes on git

  • create repository in github
  • git clone url: create folder that clones repo
  • create file(s)
  • git add :track on git/ git add *: add all
  • git commit -m "message":save new version (localy)
  • git commit -a: add and commit
  • git status: show whats going on(if you already didnt commit)...
  • git push: put on github
  • git pull: get from github to local git
  • merge conflit in pull: remove ====, 76287628cbdfa of file, then commit.
  • git log: list of all commits you made (in current branch)
  • git reset --hard : reset to preious version
  • git reset --hard origin/master: back to presious on github
  • branching: start new feature. dont change the master.
    • git branch: list all branches (only master)
    • git branch name: add a branch with name name
    • git checkout name: changes from master to name
    • git merge name: merge branch name with master
    • if want to push to github it wont work unless exist the branch in github
    • git push --set-upstream origin name: push and create branch on github.
  • remotes: version of repository elsewhere.
    • git fetch: take commits from other repo/remote and update
    • git pull: git fetch + git merge master/origin:
    • forks: Copy a repo for you.
    • pull request: merge to where you forked.
  • Github pages: go to settings, github pages enable master/branch
